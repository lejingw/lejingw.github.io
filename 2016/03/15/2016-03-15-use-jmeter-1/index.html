Use Jmeter#1

Posted at 2016-03-15 technology  jmeter 

run jmeter

non-gui mode

./jmeter -n -t ~/Desktop/facade_out_test1.jmx -l ~/Desktop/log.jtl

server mode

for distributed testing, fist run jmeter jmeter-server [-H my.proxy.server -P 8000] in server mode in remote node(s)
if you want the serer to exit after a single test has been run, then use jmeter property server.exitaftertest=true

jmeter -n -t testplan.jmx -r [-Gprop=val] [-Gglobal.properties] [-X]
where:
  -G is used to define JMeter properties to be set in the servers
  -X means exit the servers at the end of the test
  -Rserver1,server2 - can be used instead of -r to provide a list of servers to start
    Overrides remote_hosts, but does not define the property.

jmeter -n -t script.jmx -r
jmeter -n -t script.jmx -R 192.168.0.18
